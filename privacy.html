Privacy Policy

Last Updated: April 19, 2026

1. Information We Collect

Akhi Bot collects only necessary data for functionality: Server IDs, User IDs, Message Content (temporary for snipe), and Voice Activity for statistics. No personal information is collected without purpose.

2. How We Use Your Data

Data is used exclusively for bot features: Leveling, Moderation Logs, Welcome Messages, Custom Settings, and Statistics. We NEVER sell or share your data with third parties.

3. Data Storage

Data is stored securely on Supabase (EU Region) and Render.com (USA). Both providers are GDPR compliant.

4. Data Deletion

Server admins can delete all server data using !settings reset. Individual users can request data deletion via Discord DM to the developer.
